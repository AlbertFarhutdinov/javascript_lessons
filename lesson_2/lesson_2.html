Каждый N-ный пункт реализован в отдельном скрипте с именем вида script2_N.js. Потребуется консоль.

Пункт 1. Объяснение результатов кода из скрипта script2_1.js

Во второй строке инкремент в префиксной форме, значит сначала происходит увеличение значения a = 1 на единицу, а затем присвоение этого значения переменной c. В итоге выводится c = 2.

В третьей строке инкремент в постфиксной форме, значит сначала происходит присвоение значения b = 1 переменной d. а затем увеличение значения b на единицу. В итоге выводится d = 1.

После третьей строки имеем a = b = c = d = 2.

В строках 4 и 5 выражение справа от знака равно в скобках, значит операции внутри скобок производятся раньше присвоения.

Строка 4: инкремент в префиксной форме выполняется раньше сложения. Значит c = 2 + 3 = 5.

Строка 5: инкремент в постфиксной форме выполняется после сложения. Значит d = 2 + 2 = 4.

Ну и наконец после 5 строки имеем a = b = 3.
